MyFlix API

A database that will allow users to be able to register to a Movie app where they can find a certain movie within the database via Title, Genre, or Director. They are also able to update their information, add movies to their favorite movies list along with removing one, and be able to delete their account if they want to. This project mainly uses JavaScript via Node.js and MongoDB to create a database of movies and users and to perform the four operations of CRUD.
